# rsuite-snippets

> RStuie Snippets for VS Code

## Install

```sh
ext install rsuite-snippets
```

[![rsuite](./images/preview.gif)](###)



### Available Snippets

Trigger | Description
--- | ---
rsb | Default Button
rsbp | Primary Button
rsif | IconFont
rsbg | ButtonGroup
rsdd | Dropdown
rstt | Tooltip
rsp | Popover
rsnav | Nav
rsnavbar | NavBar
rsbc | Breadcrumb
rspt | Pagination
rsm | Modal
rsg | Grid
rspl | Panel
rstable | Table
rsfg | FormGroup
